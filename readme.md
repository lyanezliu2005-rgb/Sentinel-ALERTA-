# Sentinel-ALERTA- 🚀

Sistema Unificado de Alertas Vecinales de Misión Crítica para la Región Metropolitana.

## 👮‍♂️ Cobertura Institucional
* **Carabineros de Chile:** Despacho automatizado y soporte por cuadrante/comuna.
* **Paz Ciudadana:** Gestión municipal e integración de seguridad comunal.

## 📲 Canales de Ingesta Multicanal (IoT & Telco)
El sistema está diseñado para recibir y procesar alertas gatilladas mediante:
* 🎙️ **Activación por Voz:** Reconocimiento de palabras clave de pánico.
* 🏃‍♂️ **Movimientos Bruscos:** Sensores de impacto o caídas libres (Acelerómetro).
* 🚨 **Botón Físico:** Pulsación dedicada de pánico (Llaveros Bluetooth o smartphone).
* 📡 **Redundancia SMS:** Envío de alertas de texto plano cifradas en zonas de baja conectividad.
